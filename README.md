# 影刀Python模块备注工具 Web版

一个用于管理影刀RPA项目中Python模块中文备注的Web工具。

## 功能特点

- 自动检测最新的影刀RPA项目
- 支持为Python模块添加中文备注
- 备注信息集中管理，支持多项目
- 现代化的Web界面
- 即时保存提示

## 安装要求

- Python 3.6+
- Flask 2.0.1
- python-dotenv 0.19.0

## 使用方法

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python app.py
```

3. 打开浏览器访问：`http://localhost:5000`
